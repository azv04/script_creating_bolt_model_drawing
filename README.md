# script_creating_bolt_model_drawing

Этот скрипт создан на основе задания по созданию болта М64 с заданным параметрами.

Запускаете скрипт и в той же папке создается модель и чертеж.

Название файлов модели и чертежа основывается на временном промежутке с помощью библиотеки time.

Скрипт простраивает модель и чертеж отдельно друг от друга.

Сначала простраивается модель с помощью библиотеки cadquery и сохраняется в файле с форматом step в папке со скриптом.
Чертеж простраивается с помощью библиотки ezxdf и сохраняется в dxf формате. Каждый элемент чертежа: размеры, резьба, оси, расположены в своих слоях.

Чем примечательна работа:
- разбор работы библиотек cadquery и ezxdf;
- возможность создать болт со своими параметрами;
- автоматизация проектирования стандартных деталей;
